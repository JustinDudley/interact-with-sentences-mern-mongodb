NOTE!!
--Must do git commands from top-level folder
--BUT, must run program, and do npm i, from SERVER folder


To run: Go to SERVER folder, then $ node index.js
To run with nodemon, which listens for changes and reloads, go to SERVER folder, then:  $ ./node_modules/nodemon/bin/nodemon.js index.js
